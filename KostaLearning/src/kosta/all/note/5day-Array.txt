 -------------------------------------------
 
  - object relation 
  	association (use a) > aggregation (has a) > composition (consist of)
  	
  	
  	association (use a) : 사람이 렌트카를 이용하다   // local variable로 표현
  	aggregation (has a) : 사람이 스마트폰을 보유해서 사용하다  // instance variable로 표현 
  	composition (consist of) : 사람이 뇌를 보유하다 
  	                           자동차가 엔진을 보유하다
   
    배열은 객체이다. (주소값을 갖는다.) 
    
    - Array
    1) primitive data type을 저장하는 배열 
    
    int[] age; // 배열 선언 ( 배열의 타입은 int[] ) 
    age = new int[4]; // 길이 4인 배열을 생성
    age[0] = 10; 
    
    배열 선언 생성 할당을 동시에 
    ex) int[] age = {10,30,20,40}; 
    
    2) reference data type을 저장하는 배열
    // 배열 선언 및 생성
    Person p[] = new Person[3]; // 길이 3의 배열 선언 및 생성
